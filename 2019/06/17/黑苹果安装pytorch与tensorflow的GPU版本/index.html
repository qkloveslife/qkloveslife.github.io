黑苹果安装pytorch与tensorflow的GPU版本

Posted on 2019-06-17 | Edited on 2019-06-18 | In 机器学习
这篇文章只是安装思路，用于提醒自己将过程记录下来。如果安装出现问题就 Google 以下（且只能是 Google）。

安装步骤

  1. 安装 MAC_OSX 10.13.6 和 NVIDIA webdriver
  2. 安装 Anaconda(对应 Python3.7 版本)
  3. 安装 Xcode9.4 和 Command line tools
  4. 安装 CUDA10.0 for mac 和 cuDNN
  5. 安装 pytorch 与 tensorflow

安装黑苹果系统

  • 首先装配电脑
    • 我的电脑配置是 I7-8700+GTX1070
  • 安装 MAC_OSX 10.13.6
    • 目前 NVIDIA 的驱动只支持到 10.13.6，所以只能装这个版本的系统
  • 怎么做？
    • 装配电脑就不用说了
    • 安装系统找个淘宝店装一下系统，价格在 120 元左右吧。安装时候强调装 10.13.6 版本，并安装 NVIDIA webdriver。

安装 Anaconda

  • 下载 Anaconda并安装

安装 Xcode9.4 和 Command line tools

  • 下载 Xcode9.4 并安装
    • 10.13.6 好像只支持到 9.4 吧，高版本可能要求 10.14。
  • 下载 Command line tools
    • 既然装了 Xcode9.4 就必须安装对应版本的 Command line tools
  • 怎么做？
    • 下载 Xcode9.4，AppStore 只能安装最新版 Xcode。解压后拖拽到“应用程序”文件夹里就行了。
    • 下载 Command line tools，版本要对应，否则进行后面步骤时会出错。

安装 CUDA10.0 和 cuDNN

  • 下载 CUDA10.0 并安装
  • 下载 cuDnn 并安装
  • 怎么做？
    • 下载 CUDA10.0 点击 “Download(1.8G)” 下载。安装教程
    • 下载 cuDNN。这里需要你有 NVIDIA 开发者账号，注册一个就好了。登陆后可以下载对应 CUDA10 版本的 cuDNN 了。安装教程

安装 pytorch 与 tensorflow

  • 下载 GPU 版本的 pytorch 和 tensorflow
  • 怎么做？
    • 到这个页面找对应版本的 pytorch，以及下面给出的 tensorflow 和 MxNet，然后用 pip 安装（安装前用 conda 创建虚拟环境）。
